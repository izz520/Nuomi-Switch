# Product

## Register

product

## Users

Nuomi Switch serves developers and power users who manage multiple Codex accounts locally. They use it while working on their machine, often needing to import credentials, switch the active Codex auth file, inspect quota, and recover from local configuration issues without hand-editing files.

## Product Purpose

Nuomi Switch is a local-first desktop app for managing Codex accounts, OAuth credentials, API key accounts, quota state, sessions, settings, and logs. Success means users can trust the app to make sensitive account operations clear, reversible where possible, and easier than manual config edits.

## Brand Personality

Calm, precise, trustworthy. The interface should feel like a focused desktop utility: quiet enough for repeated use, clear about risk, and polished without getting in the user's way.

## Anti-references

Avoid marketing-page drama, decorative dashboards, oversized hero layouts, and flashy modal treatments. Avoid interfaces that make credential handling feel casual, noisy, or ambiguous.

## Design Principles

- Keep account operations explicit and scannable.
- Preserve local-first trust by making paths, backup behavior, and sensitive actions clear.
- Prefer familiar desktop utility patterns over novelty.
- Use visual polish to reduce friction, not to decorate.
- Keep density practical for users managing many accounts.

## Accessibility & Inclusion

Target WCAG 2.1 AA for contrast, keyboard access, focus visibility, and reduced-motion support. Credential and error states should never rely on color alone.
